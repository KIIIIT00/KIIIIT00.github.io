オセロゲームアプリ

開発期間: 2022年10月 - 2023年11月

Android Studio Java

概要

Android Studioを用いてCPU対戦機能，対面対戦機能を備えたオセロゲームアプリ

使用技術

  • バックエンド: Java
  • 開発環境: Android Studio

主な機能

  1. CPU対戦機能: ユーザは、強さが3段階のCPUと対戦することができる。
  2. 対面対戦機能: ユーザは、1つのスマートフォンを通して、オセロを楽しむことができる。

学んだこと・工夫した点

Android Studioで、オセロのゲームAIの作成について学ぶことができた。

Android Stuidoを用いた初めてのアプリ開発で，わからないことがたくさんあったが，あきらめずに、自分で調べて一通り形にすることができた。
